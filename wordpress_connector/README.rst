Configuration
=============

Add the following parameters to your Odoo configuration file:

-  ``wp_sftp_host`` : the server url of your wordpress sftp (ex:
   wp.localhost.com)
-  ``wp_sftp_user`` : a sftp user of your wordpress server
-  ``wp_sftp_pwd`` : the sftp password
