Nach einer Anzahlung von CHF 100 wird die Anmeldung gültig. Ich bekomme dafür eine Rechnung von 4M zugesendet.

Ich bin mir bewusst, dass ich neben der Spendensammlung die normalen Teilnehmerkosten bezahlen werde.

Um an einem Muskathlon teilnehmen zu können, wird ein Fragebogen zur körperlichen Gesundheit ausgefüllt und es wird empfohlen, sich durch den Hausarzt untersuchen zu lassen.

Ich bin mir bewusst das ich für dieses Projekt CHF 10.000 CHF Sponsorengelder sammeln werde.

Ich werde mein Möglichstes tun um dieses Ziel zu erreichen.

10% der Sponsorengelder werden von 4M CH für den organisatorischen Aufwand des Muskathlons verwendet.

Die Registrierungsgebühr von CHF 100 sind die minimalen Annullierungsgebühren.

Für eine Annullierung innert vier Monaten vor dem Start des Muskathlon werden 25% der Reisekosten veranschlagt. Diese Anzahlung leisten alle Teilnehmenden vier Monate vor dem Start des Muskathlons.

Für eine Annullierung innert drei Monaten vor dem Start des Muskathlon werden 50% der Reisekosten veranschlagt. Diese Anzahlung leisten alle Teilnehmenden drei Monate vor dem Start des Muskathlons.

Für eine Annullierung innert zwei Monaten vor dem Start des Muskathlon werden 100% der Reisekosten veranschlagt. Den vollen Reisebetrag bezahlen alle Teilnehmenden zwei Monate vor dem Start des Muskathlons.
